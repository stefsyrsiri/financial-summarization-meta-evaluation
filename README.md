# Financial Narrative Summarization Evaluation (In-progress)

## Overview

In this project we evaluate automatic summarization evaluation methods on the summaries of Greek annual reports.

## Project structure

```sh
thesis/
│── data/                         # Source documents, gold summaries, candidate summaries (generated by pipelines/generate.py)
│── evaluation_methods/           # Evaluation methods cloned repos
│── results/                      # Evaluation results from pipelines/evaluate.py
│── src/
|   ├── modules/
│       ├── data_collector.py     # Data collection class
│       ├── summary_corruptor.py  # Noise insertion class
│       ├── summary_generator.py  # Candidate summaries generator class
│       ├── summary_evaluator.py  # Evaluation class
|   ├── pipelines/
│       ├── collect.py            # Data collection pipeline
│       ├── generate.py           # Candidate summaries generation pipeline
│       ├── evaluate.py           # Evaluation pipeline
|   ├── utils/                    # TODO: Helper functions
│   ├── main.py                   # Main script
│── .gitattributes
│── .gitignore
│── analysis.ipynb                # Noise insertion demonstration and evaluation findings
│── README.md
│── requirements.txt
│── setup.cfg

```

## Installation

1. Clone the repository:

    ```sh
    git clone https://github.com/stefsyrsiri/thesis.git
    cd thesis
    ```

2. Install dependencies:

    ```sh
    pip install -r requirements.txt
    ```

## Usage

Run the script:

```sh
python src/main.py
```
